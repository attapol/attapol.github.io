Thai Literature Corpus (TLC)

A corpus of sectionized Thai literature texts from Vajirayana Digital Library

by Jitkapat Sawatphol (Arts, Chulalongkorn)

Usage example

Extract the tlc file then import each document with:

doc = json.loads(open(filename).read())

You will get a python list of dictionaries containing chapter number, title, and text of each section of each text.

Then the text (list of lines) of each chapters can be accessible with:

doc['text']

You can also get the title of each chapter with['ch_num']

doc['title']

and chapter number with

doc['ch_num']
